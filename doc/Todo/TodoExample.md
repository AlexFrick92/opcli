# Пример создания Todo файла

Todo это:
- название задачи
- описание задачи
- шаги необходимые для выполнения задачи
- итоги работы - резюме

## Расположение файлов
Файлы создаются в этой директории - doc/Todo/

## Формат имени файла:

`Todo_[название задачи].md`

Пробелы не допускаются.

## Формат файла:
```
# [Название задачи]

## Overview
[Краткое описание задачи]

## Step-by-step 
[Шаги выполения задачи]

## Progress

## Summary
[Итоги работы]
```

## Формат раздела Step-by-step:

```
1. [описание шага 1]
2. [описание шага 2]
...
10. [Описание шага 10]
```
Чекбоксы не допускаются.

## Формат раздела Progress

Основные функциональные и технические нюансы выполнения каждого шага.
По этом разделу отслеживается фактическое выполнение задачи.

## Формат раздела Summary

Краткое описание выполнения всей задачи.
Составляется после завершения последнего шага.
